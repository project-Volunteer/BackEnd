[[Notice-Comment-API]]
== Notice-Comment API

=== 봉사 모집글 공지사항 댓글 등록
==== 발생 가능 예외
|===
|HTTP 상태코드|메시지
|400| - 해당 모집글 정보를 찾을 수 없습니다. +
- 봉사 모집글 활동 가능 기간이 종료되었습니다. +
- 해당 공지사항 정보를 찾을 수 없습니다. +
- 입력한 데이터가 요구 사항을 충족하지 않습니다. 필수 항목을 확인하고 다시 시도해주십시오.
|401| - 인증이 필요합니다. 로그인 후 다시 시도하십시오.
|403| - 봉사 팀원에게만 공개되는 정보입니다. 팀원 참여 후 이용해주시길 바랍니다.
|500| -
|===
