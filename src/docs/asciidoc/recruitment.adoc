== Recruitment API

=== 봉사 모집글 등록
==== 발생 가능 예외
|===
|HTTP 상태코드|메시지
|400| - 입력한 데이터가 요구 사항을 충족하지 않습니다. 필수 항목을 확인하고 다시 시도해주십시오.
|401| - 인증이 필요합니다. 로그인 후 다시 시도하십시오.
|500| -
|===
