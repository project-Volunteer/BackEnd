== Recruitment Management API

=== 봉사 팀원 가입 신청
==== 발생 가능 예외
|===
|HTTP 상태코드|메시지
|400| - 해당 모집글 정보를 찾을 수 없습니다. +
- 봉사 모집 기간이 종료되었습니다. +
- 모집인원이 마감되었습니다. +
- 중복 신청을 할 수 없습니다. 신청 현황을 확인해주십시오.
|401| - 인증이 필요합니다. 로그인 후 다시 시도하십시오.
|500| -
|===
