== Schedule Management API

=== 봉사 일정 참여
==== 발생 가능 예외
|===
|HTTP 상태코드|메시지
|400| - 해당 일정 정보를 찾을 수 없습니다. +
- 일정 모집 기간이 종료 되었습니다. +
- 모집인원이 마감되었습니다. +
- 중복 신청을 할 수 없습니다. 신청 현황을 확인해주십시오. +
-  사용자의 팀원 정보를 찾을 수 없습니다.
|401| - 인증이 필요합니다. 로그인 후 다시 시도하십시오.
|403| - 봉사 팀원에게만 공개되는 정보입니다. 팀원 참여 후 이용해주시길 바랍니다.
|500| -
|===
